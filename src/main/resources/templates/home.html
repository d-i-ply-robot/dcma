ALTApro - DCMA Program Checker

ALTApro is an automated construction schedule health check. It uses metrics to assess and provide feedback to the user to help make more robust and effictive construction programs.

This process will only test program for symptoms of poorly structured programs. It will provide an objective report on potential structural issues that once corrected should tend to make programs more robust. The process will not report on unrealistic durations or flawed construction sequencing logic. DCMA 14 Point Check The DCMA 14-point checklist guideline that has been established to objectively assess programs. More background information on the DCMA assessment can be found at:
